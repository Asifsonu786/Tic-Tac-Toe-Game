# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

## 📌 Overview

This project is a browser-based Tic Tac Toe game where two players take turns placing "X" and "O" on a 3×3 grid. The game automatically detects winners, handles draw situations, and provides options to start a new game or reset the current game.

## ✨ Features

- Two-player gameplay (X vs O)
- Automatic winner detection
- Draw game detection
- Reset game functionality
- New game option
- Responsive and clean user interface
- Interactive button hover effects

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 How to Run

1. Clone this repository:

```bash
git clone https://github.com/your-username/tic-tac-toe.git
```

2. Open the project folder.

3. Run the project by opening `index.html` in any web browser.

No additional installation or dependencies are required.

## 🎯 Game Rules

1. Player O starts first.
2. Players take turns marking empty cells.
3. The first player to align three symbols horizontally, vertically, or diagonally wins.
4. If all cells are filled and no player wins, the game ends in a draw.
